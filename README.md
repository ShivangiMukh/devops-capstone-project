devops-capstone-project

![Build Status](https://github.com/ShivangiMukh/devops-capstone-project/actions/workflows/ci-build.yaml/badge.svg)


This project involves the development and extension of a Customer Account Microservice designed to manage user data for an e-commerce platform. The service is architected as a standalone microservice to ensure modularity and scalability, exposing a robust RESTful API to facilitate seamless communication with other internal services.
